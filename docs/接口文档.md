# 接口文档

## 基础

### GET `/api/health`

响应：

```json
{
  "ok": true,
  "service": "knowledge-agent",
  "version": "0.1.0",
  "llm_provider": "modelscope",
  "llm_model": "deepseek-ai/DeepSeek-V3.2",
  "llm_key_configured": true
}
```

## 教材解析

### POST `/api/upload`

请求：`multipart/form-data`，字段名为 `files`，可一次上传多个 PDF/Markdown/TXT/DOCX。

响应：

```json
{
  "files": [
    {
      "filename": "03_生理学.pdf",
      "status": "accepted",
      "textbook_id": "book_03419897"
    }
  ]
}
```

### GET `/api/textbooks`

响应：

```json
[
  {
    "textbook_id": "book_03419897",
    "filename": "03_生理学.pdf",
    "title": "03_生理学",
    "file_format": "pdf",
    "total_pages": 489,
    "total_chars": 621084,
    "chapter_count": 13,
    "parse_status": "done",
    "error_message": null
  }
]
```

响应：

```json
[

    "page_start": 1,
    "page_end": 28,
    "char_count": 31240,
    "content": "# 第一章 绪论\n..."
  }

```

## 知识图谱

### POST `/api/graph/build/{textbook_id}`

响应：

```json
{
  "started": true,
  "textbook_id": "book_03419897",
  "chapters": 13
}
```

### GET `/api/graph/{textbook_id}`

响应：

```json
{
  "textbook": {
    "id": "book_03419897",
    "title": "03_生理学",
    "filename": "03_生理学.pdf"
  },
  "nodes": [
    {

    {
      "source": "node_a",
      "target": "node_b",

      "description": "理解 B 需要先掌握 A"
```

### GET `/api/graph`

返回所有教材的合并图谱，节点包含 `textbook_id`、`textbook_title`、`frequency`，用于教材来源颜色和频次大小映射。

## 跨教材整合

### POST `/api/integrate/run`

请求：

```json
{
  "textbook_ids": []
}
```

响应：

```json
{
  "started": true,
  "run_id": "run_abc123",
  "textbook_ids": []
}
```

响应：

```json
{
  "run_id": "final_7book_acceptance",
  "status": "done",
  "stats": {
    "original_textbooks": 7,
```

### PATCH `/api/integrate/decisions/{decision_id}`

请求：

```json
{
  "action": "keep",
  "reason": "教师认为该知识点应单独保留",
  "confidence": 0.97
}
```

```json
{
  "updated": true,
  "decision": {
    "decision_id": "final_7book_acceptance_merge_0001",
    "action": "keep",
    "reason": "教师认为该知识点应单独保留..."
  },
  "stats": {
    "decisions_merge": 23,
    "decisions_keep": 237,
    "decisions_remove": 0
  }
}
```

## 对话 Agent

### POST `/api/chat`

请求：

```json
{
  "session_id": "demo",
  "message": "不要合并 [[炎症]] 和 [[炎症反应]]"
}
```

响应：

```json
{
  "session_id": "demo",
  "reply": "已更新：final_7book_acceptance_merge_0001 已由 merge 调整为 keep。",
  "tool_name": "modify_decision",
  "history": []
}
```

## 报告

### GET `/api/reports`

列出可在前端“报告”Tab 中查看的 Markdown 文档。

### GET `/api/reports/{report_id}`

返回 `需求分析`、`系统设计`、`Agent 架构说明`、`整合报告`、`P2 技术报告` 等 Markdown 正文。
